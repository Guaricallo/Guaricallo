# 📘 Guía Rápida de Markdown para README en GitHub

## 📌 Encabezados
```markdown
# Título 1
## Título 2
### Título 3
```

## ✏️ Formato de Texto
```markdown
*Texto en cursiva*  
**Texto en negrita**  
~~Texto tachado~~  
`Texto en línea`
```

## 🔗 Enlaces
```markdown
[Texto del enlace](https://ejemplo.com)
<https://ejemplo.com>
```

## 🖼️ Imágenes
```markdown
![Texto alternativo](https://ruta-de-la-imagen.jpg)
```

## 📝 Listas
### Listas ordenadas
```markdown
1. Primer ítem
2. Segundo ítem
```

### Listas no ordenadas
```markdown
- Ítem
* Otro ítem
+ Tercer ítem
```

## 🔢 Código
### En línea
```markdown
`código`
```
### Bloques
<pre>
```python
def saludar():
    print("Hola")
```
</pre>

## 📋 Tablas
```markdown
| Columna 1 | Columna 2 |
|-----------|-----------|
| Dato 1    | Dato 2    |
```

## 📌 Citas
```markdown
> Esta es una cita
>> Cita anidada
```

## ✅ Listas de Tareas
```markdown
- [x] Hecho
- [ ] Pendiente
```

## ⬇️ Separadores
```markdown
---
***
___
```

## 🧭 Referencias de enlaces
```markdown
[Google][1]
[1]: https://www.google.com
```

## 🧩 HTML en Markdown
```html
<b>Texto en negrita</b>
<img src="https://ruta.com/img.jpg" width="100">
```

---

> 🛠️ **Consejo:** Usa Markdown para mantener tu documentación limpia, legible y profesional en GitHub.
